### 디렉토리 구조
````
├───java
│   └───kr.personal.js
│        │   Example.java // 테스트를 위해 사용한 코드이며 method 사용의 이해를 돕기 위한 클래스입니다.
│        │   Main.java // 테스트 실행부입니다.
│        │
│        ├───todo1 // 1번 문제 풀이가 담겨있는 디렉토리입니다.
│        │       CategoryDatabase.java // Category 를 자료구조인 StructuredData 를 사용하기 위한 class 입니다.
│        │       StructuredData.java // 자료 구조화를 위한 자료구조 class 입니다.
│        │
│        └───todo2 // 2번 문제 풀이가 담겨있는 디렉토리입니다.
│                CoinCombination.java // 동전 조합 문제 풀이를 위한 class 입니다.
│
└───resources
````
